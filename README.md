# OnyxVK
A Vulkan-Based engine for game development

To compile, make sure you have the following installed (for Arch):
```
vulkan-tools
vulkan-loader-devel
vulkan-headers
vulkan-validation-layers

Above can be installed with vulkan-devel

glfw-wayland  (or glfw-x11)
glm
shaderc
libxi
libxxf86vm
```
